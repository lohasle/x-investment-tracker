# X 投资情报跟踪系统

> 基于 X.com 的长期投资情报跟踪与沉淀系统

## 简介

本仓库用于沉淀每日投资情报日报、高价值账号画像、主题研究等**生成结果**。

**不包含**任何内部实现、配置、凭据或自动化细节。

## 目录结构

```
/daily/          每日日报 (YYYY-MM-DD.md)
/weekly/         周报 (YYYY-Wxx.md)
/monthly/        月报 (YYYY-MM.md)
/accounts/       账号分层与排名
/profiles/       账号画像
/themes/         主题研究页
/raw/            原始记录
/docs/           站点文档
```
